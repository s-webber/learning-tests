[introduction]
= Introduction
This is an example of documentation for a RESTful API that is output to `target/generated-docs` as part of a `mvn install` and consists of:

. Static text manually maintained in `src/main/asciidoc/example.adoc`.
. Swagger documentation that is generated by `springfox-staticdocs` as part of executing `src/test/java/com/example/Swagger2MarkupTest.java`.
. Example HTTP requests and responses generated by Spring REST Docs as part of executing `src/test/java/com/example/ExampleTest.java`.
